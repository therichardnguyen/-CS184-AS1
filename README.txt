Playform:
MAC
 
Features: 

- can draw multiple polygons and lines. Everything will be saved into one .obj file.
- TAB key to cycle through 7 different colors: {white, red, blue, green, yellow, magenta, cyan}
- each 'polygon' can have it's own color which can be changed while building it, but not after.
- BACKSPACE to clear the screen and start a new .obj file.
- '=' key to increase the width of every line of the polygons
- '-' key to decrease the width of every line of the polygons
- SPACE to make the current 'polygon' a GL_LINE_STRIP instead
- 'z' to erase the current 'polygon' that's being drawn 
- .obj file also outputs lines: 'l v1 v2'
